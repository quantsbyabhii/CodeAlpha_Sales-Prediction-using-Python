# 📈 Sales Prediction using Python

## 📌 Project Overview
This project focuses on predicting sales using Python and Machine Learning techniques by analyzing advertising and business-related data.

The project covers data preprocessing, exploratory data analysis (EDA), visualization, model training, evaluation, and sales forecasting.

---

## 🎯 Objectives
- Analyze sales-related data
- Perform data cleaning and preprocessing
- Explore relationships between variables
- Build a Machine Learning prediction model
- Forecast future sales
- Generate business insights

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

The dataset contains business and advertising information used to predict sales outcomes.
